Understanding Why Users Ignore Data Breach Warnings

This project studies how users react to data breach notifications and why many people ignore security warnings even after being alerted.

Problem Statement

Many users receive data breach alerts but do not take action such as changing passwords or enabling security features. This increases their risk. The project aims to understand the behavioral reasons behind this inaction.

Literature Review

Research shows that users experience warning fatigue and often ignore repeated security alerts. Message clarity and design influence how seriously users respond.

Research Gap

Existing tools detect breaches but do not analyze how users respond after being notified. This project focuses on understanding student-level behavior.

Methodology

Data Collection: Anonymous student survey to measure awareness and response behavior.

Analysis: Responses categorized based on awareness, action taken, and risk perception. Basic statistical analysis will be used.
